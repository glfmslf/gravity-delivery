# 反重力快递

一个网页动作挑战小游戏。玩家驾驶绑着纸箱的小型快递飞行器，在城市楼道中赶单，通过翻转重力躲避上下通道障碍，贴近绿色门牌完成全部投递，拾取黄色时间包延长倒计时，并在时间耗尽前送达终点。

## 当前阶段

MVP 试玩阶段：

- 已实现反重力移动、障碍、投递门牌和时间包
- 已实现两个手工关卡，成功后进入下一关
- 已实现成功/失败状态、重开流程和基础音效
- 已准备 Node 零依赖本地启动脚本

## 运行方式

推荐使用本地静态服务运行：

```bash
npm start
```

然后打开 `http://localhost:4173`。

不要直接用浏览器打开 `index.html`。当前页面通过 ES module 加载 `src/*.js`，在 `file://` 协议下容易被浏览器拦截。

测试命令：

```bash
npm test
```

## 试玩方式

- 按 `Space`、`Enter`、点击画布或点击“开始 / 翻转”按钮开始。
- 配送中按 `Space`、点击画布或点击按钮翻转重力。
- 贴近绿色门牌完成全部投递。
- 拾取黄色时间包加时，避开橙红色障碍。
- 到达终点但漏送订单，或时间归零，都会失败。
- 成功或失败后按 `Space` / `Enter`，或点击画布/按钮，可直接重开。
- 成功后再次操作会进入下一关，失败后会重开当前关。
