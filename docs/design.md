# 反重力快递设计文档

## 项目定位

《反重力快递》是一款轻量网页动作挑战小游戏。玩家驾驶一台绑着纸箱的小型快递飞行器，在城市楼道、管道和招牌之间穿梭，在倒计时结束前把包裹送到终点。

第一阶段目标是做出一个能玩、能展示、能继续迭代的 MVP，而不是一次性完成完整商业游戏。

## 目标玩家体验

玩家应该在 10 秒内理解规则，在 1 分钟内进入紧张状态，并在失败后愿意立刻重开。

核心感受：

- 切换重力带来明显的方向反转反馈。
- 左右移动让玩家有操作余地，而不是只靠一次按键赌命。
- 时间机制让主题成立：玩家不是单纯活下去，而是在赶单。
- 撞击扣时间让第一版更宽容，方便调试和试错。

## MVP 范围

第一版包含：

- 一个网页 Canvas 游戏页面
- 一台可控制的快递飞行器
- 左右键横移
- 空格切换重力
- 一个限时送达关卡
- 障碍物
- 时间包
- 撞击扣时间
- 到达终点成功
- 时间归零失败
- 重开按钮

第一版不包含：

- 多关卡
- 排行榜
- 账号或存档
- 商店和成长系统
- 复杂剧情
- 移动端深度适配

## 核心规则

### 操作

- `ArrowLeft`：向左移动
- `ArrowRight`：向右移动
- `Space`：切换重力方向

### 胜利

玩家在时间归零前到达关卡终点，配送成功。

### 失败

倒计时归零，配送失败。

### 时间变化

- 初始时间：60 秒
- 拾取时间包：增加时间
- 撞击障碍：扣除时间，并短暂无敌或冷却，避免连续扣除

具体数值在开发中调参确定。

## 视觉方向

风格是“纸箱快递 + 城市楼道”：

- 主角像小型快递飞行器，绑着纸箱。
- 背景是城市楼道、楼层、管道、招牌。
- 色彩使用暖色纸箱、深色楼道、青绿色功能元素和橙红色危险元素。
- 第一版优先用 Canvas 绘制几何图形，后续再替换为图片或动效资源。

## 技术方案

第一阶段使用原生前端：

- `HTML` 提供页面结构
- `CSS` 提供布局和视觉样式
- `JavaScript` + `Canvas` 实现游戏循环、绘制和交互

暂时不引入 React、构建工具或游戏引擎。这样启动成本低，调试直接，适合快速验证玩法。

## 模块职责

`src/main.js` 负责连接 DOM、创建游戏实例和绑定重开按钮。

`src/game.js` 负责游戏主循环、状态流转、倒计时、胜负判断和 UI 更新。

`src/input.js` 负责键盘输入状态。

`src/player.js` 负责玩家位置、速度、重力方向和边界处理。

`src/level.js` 负责关卡长度、速度、障碍物和拾取物数据。

`src/renderer.js` 负责 Canvas 绘制。

## 后续扩展方向

玩法稳定后，可以考虑：

- 多个手工设计关卡
- 更细的碰撞反馈
- 音效和简单背景音乐
- 连击或准时送达评分
- 移动端触摸控制
- Phaser 或 PixiJS 重构
- 关卡编辑器
